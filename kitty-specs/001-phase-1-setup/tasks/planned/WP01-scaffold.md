Scaffold the project from the Phaser 3 + Vite + TypeScript template.

## Steps
1. Run `npx degit phaserjs/template-vite-ts . --force`
2. Run `npm install`
3. Verify `npm run dev` works with the default template

## Acceptance
- Template files present (src/, index.html, package.json, tsconfig.json, vite/)
- `npm run dev` starts without errors
- Default Phaser template renders in browser
